# AdminGolazoPro

Una aplicación Android moderna para gestionar datos de fútbol, desarrollada con **Kotlin** y **Jetpack Compose**.

## 📱 Características Principales

- **Arquitectura Moderna**: Construida con Compose, Hilt y Room Database
- **Autenticación Segura**: Integración con biometría y Firebase
- **Gestión de Datos Offline**: Room Database para persistencia local
- **Notificaciones Push**: Firebase Cloud Messaging
- **Multientorno**: Configuración separada para DEV y PROD
- **APIs Externas**: Integración con Rick and Morty API y JSON Placeholder

## 🛠️ Stack Tecnológico

### Lenguaje y Framework
- **Kotlin** - Lenguaje principal
- **Jetpack Compose** - UI moderna y declarativa

### Inyección de Dependencias
- **Hilt** - Gestión de dependencias
- **KSP** - Kotlin Symbol Processing para compilación rápida

### Persistencia de Datos
- **Room Database** - Base de datos local
- **Firebase** - Analytics y Messaging

### Networking
- **Retrofit** - Cliente HTTP
- **OkHttp** - Logging y interceptores
- **Kotlinx Serialization** - Serialización JSON

### UI y Navegación
- **Material 3** - Componentes de diseño Material
- **Google Fonts** - Tipografía
- **Coil** - Carga de imágenes
- **Material Icons Extended** - Iconografía ampliada
- **Navigation Compose** - Navegación en Compose

### Seguridad
- **Biometric** - Autenticación biométrica
- **Security Crypto** - Almacenamiento seguro de credenciales

### Tasks & Work
- **WorkManager** - Tareas en background

## 📋 Requisitos Previos

- Android Studio Koala o superior
- JDK 21
- Android SDK compileSdk 36
- minSdk 26
